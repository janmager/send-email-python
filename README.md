## 🐍 send-email-python

Simple python script with prebuild modules to sending emails.

Just set this lines of code to your own settings

```sh
# Settings
yourAdress = 'your email'
yourPassword = 'your password'
receiverEmail = 'receiver email'
subject = 'subject'
content = 'content'
smtp = 'your.smtp.com'
port = 111

```
